Sobre o HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas, como if e else, por exemplo. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

A estrutura das tags consistem em:

  1. Abertura;
  2. Conteúdo;
  3. Fechamento.

Porém, também existem tags que não precisam de conteúdo, consequentemente não precisam ser fechadas, como as tags de imagem ( < img > ) e quebra de linha (< br >), por exemplo.
